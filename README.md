This is a small HTML program I wrote for manipulating text.

* The whole tool is one HTML file that you open in your browser.
* Flow chart interface (similar to Blender's shader editor) allows for complex behavior you had to program before.
* You can save setups by right clicking in your browser and saving the page to disk; everything will still work.
* No character limits.
